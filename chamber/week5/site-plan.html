About the Application

This web app allows monitoring changes on municipal bidding pages and sending email alerts.

Target Users

  • Small businesses and suppliers
  • Journalists and researchers
  • Citizens interested in transparency

Main Features

  • Municipality list with portal links
  • Change detection and update tracking
  • Daily feed of today's changes
  • Search and filter by state, municipality, or type
  • User registration and login
  • Email notifications
  • Favorites / subscriptions
  • Change history log
  • Responsive design
  • Error handling for unavailable municipal sites
